# 🌍 Multilingual Nadesiko3 - English & Kid-Friendly Programming

## 🌐 English Support for Foreign Users

**This is AI modified!** Now Nadesiko3 supports English programming commands, making it accessible to international users who find Japanese difficult.

### 🚀 English Commands

**Basic Operations:**
```nako3
PRINT "Hello World!"           # Display text
DISPLAY "Welcome!"              # Show message
SHOW "Result: {value}"          # Show with formatting
CALCULATE "2+2"                # Mathematical calculation
```

**Mathematics:**
```nako3
ADD 5 and 3                    # Addition (5 + 3)
SUBTRACT 3 from 10             # Subtraction (10 - 3)
MULTIPLY 4 by 6                # Multiplication (4 × 6)
DIVIDE 12 by 4                 # Division (12 ÷ 4)
```

**Variables:**
```nako3
CREATE_VARIABLE 100 as "score"  # Create variable
MAKE_VARIABLE "Hello" as "message"  # Alternative syntax
```

**Control Flow:**
```nako3
IF condition THEN block         # Conditional
REPEAT 5 times block           # Loop
LOOP_UNTIL condition block      # While loop
```
