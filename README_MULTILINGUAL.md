## 🌟 Language Comparison

### 📝 **Basic Commands**

| Japanese | English | Description |
|----------|----------|-------------|
| 「こんにちは」を表示 | "Hello" display | Display text |
| 5と3を足して | 5 and 3 add | Addition |
| 4と6を掛けて | 4 and 6 multiply | Multiplication |
| 10から2を引いて | 10 subtract 2 | Subtraction |
| 3回「楽しい！」と繰り返して | 3 repeat "Fun!" | Repeat |
| Aに10を代入する | assign 10 to A | Variable assignment |

### 🎮 **Programming Examples**

#### 🇯🇵 **Japanese Example**
```nako3
「こんにちは、なでしこ3AI！」と表示
5と3を足して
3回「楽しい！」と繰り返して
Aに10を代入する
「Aの値は{A}です」と表示
```

#### 🇺🇸 **English Example**
```nako3
"Hello, Nadesiko3AI!" display
5 and 3 add
3 repeat "Fun!"
assign 10 to A
"The value of A is {A}" display
```
