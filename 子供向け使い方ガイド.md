# 🌟 なでしこ3AI 子供向け使い方ガイド 🌟

## 🎯 これがなでしこ3AIだよ！

なでしこ3AIは、**日本語でプログラミング**できる魔法のツール！
難しい英語や記号はいらないよ。いつも使っている日本語で、
アニメやゲームみたいに楽しくプログラミングできるんだ！

---

## 🚀 すぐに始めよう！

### 方法1: Webで始める（一番簡単！）

1. **nadesiko3ai_web.html** をダブルクリック
2. ブラウザが開いたら、すぐに始められる！
3. テキストボックスに日本語で命令を書こう

### 方法2: コマンドプロンプトで始める

1. **nadesiko3ai_starter.js** をダブルクリック
2. 黒い画面が出てきたら、日本語で命令を入力
3. Enterキーを押すと実行される！

---

## 💡 最初に試してみよう！

### 📝 こんにちはを表示しよう

```
「こんにちは、なでしこ3AI！」と表示
```

```

```

