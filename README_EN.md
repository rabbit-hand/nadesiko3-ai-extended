## What is Nadesiko3 - AI Enhanced Version?

**This is AI modified!** Nadesiko3 is a Japanese programming language. Based on JavaScript/TypeScript, it runs on various environments including PCs, smartphones, and tablets. Japanese programming languages are characterized by being easy to read and understand, making it fun to learn programming even for beginners.

**🚀 AI Enhanced Features:**
- **Python-level advanced features fully implemented**
- **Data Science, Machine Learning, Deep Learning capabilities**
- **Natural Language Processing, Time Series Analysis features**
- **IoT Hardware Control, Web API integration**
- **🌍 Multilingual Support**: English programming + Kid-friendly mode
- **🎈 Kids Won't Cry**: Animal sounds, emoji reactions, fun commands
- **Ultimate Japanese programming language for education, research, and practical use**

## 🌍 Multilingual Programming

### English Mode
```nako3
# Set language mode
「言語モード設定」に「english」

# Basic English commands
PRINT "Hello World!"
DISPLAY "Welcome to Nadesiko3!"
CALCULATE "2+2"
ADD 5 and 3
MULTIPLY 4 by 6
CREATE_VARIABLE 100 as "score"
koko
```

### Kid-Friendly Mode (かんたんモード)
```nako3
# Switch to kid mode
「かんたんモード」

# Fun commands for kids
おはよう                          # Good morning! ☀️
わんわん                          # 🐕 Woof woof!
かずあそび 5と3                   # 5 + 3 = 8 🎯
おんがくを「ド」                   # 🎵 Do~
ふくふくを「AIさん」                # 🎈 Decorate text!
koko
```

### Mixed Language Programming
```nako3
# Mix Japanese, English, and kid-friendly commands
「おはよう」と表示                    # Japanese
PRINT "Hello World!"                 # English
「かんたんモード」                    # Switch to kid mode
わんわん                             # Kid-friendly command
ADD 10 and 5                         # English math
「おやすみ」と表示                    # Japanese goodbye
koko
```

## 🎈 Kid-Friendly Features

### Fun Commands for Children
- **Greetings**: おはよう, こんにちは, こんばんは, おやすみ
- **Animal Sounds**: わんわん (dog), にゃんにゃん (cat), ぴよぴよ (bird), ぽっぽ (train)
- **Number Games**: かずあそび (addition game), たしざん (addition), ひきざん (subtraction), かけざん (multiplication)
- **Music Play**: おんがく (musical notes: ドレミファソラシ)
- **Fun Activities**: えらんで (random pick), くじびき (lottery), ことばあそび (word play), ふくふく (decorate)
- **Creative**: ものがたり (storytelling), おえかき (drawing), あそぼう (let's play)

## 🌐 English Commands Reference

### Basic Operations
```nako3
PRINT "text"                    # Display text
DISPLAY "message"                # Show message  
SHOW "result"                   # Show with formatting
CALCULATE "expression"           # Mathematical calculation
koko
```

### Mathematics
```nako3
ADD 5 and 3                    # Addition (5 + 3)
SUBTRACT 3 from 10              # Subtraction (10 - 3)
MULTIPLY 4 by 6                # Multiplication (4 × 6)
DIVIDE 12 by 4                 # Division (12 ÷ 4)
koko
```

### Variables and Control
```nako3
CREATE_VARIABLE 100 as "score"  # Create variable
IF condition THEN block            # Conditional statement
REPEAT 5 times block            # Loop 5 times
koko
```

## 🔧 Installation and Setup

### Install via npm
```bash
npm install -g nadesiko3
```

### Run Commands
```bash
# Command line execution
cnako3 -e "「こんにちは」と表示。"
# Execute source file
cnako3 hello.nako3
koko
```

### GitHub Installation
```bash
git clone https://github.com/rabbit-hand/nadesiko3ai.git
cd nadesiko3ai
npm install
npm run build
koko
```

## 📖 Examples

### Data Science Example
```nako3
# Load data science plugin
「言語モード設定」に「japanese」

# Create sample data
データ = [[1,2,3], [4,5,6], [7,8,9]]
平均値 = データを平均
標準偏差 = データを標準偏差

「平均値: {平均値}」と表示
「標準偏差: {標準偏差}」と表示
koko
```

### Machine Learning Example
```nako3
# Machine learning with English commands
「言語モード設定」に「english」

# Create training data
training_data = [[1,2], [3,4], [5,6]]
labels = [3, 7, 11]

# Train linear regression
model = LINEAR_REGRESSION training_data labels

# Make prediction
prediction = PREDICT model [7,8]
DISPLAY "Prediction: {prediction}"
koko
```

### Web API Example
```nako3
# Web API integration
「言語モード設定」に「english」

# Get weather data
weather = HTTP_GET "https://api.weather.com/data"
temp = JSON_GET weather "main.temp"

DISPLAY "Temperature: {temp}°C"
koko
```
