# 🌟 Nadesiko3AI Kids User Guide (English) 🌟

## 🎯 Welcome to Nadesiko3AI!

Nadesiko3AI is a magical tool that lets you **program in Japanese**!
No difficult English or symbols needed. You can program as easily as if you were playing games or watching anime, all using your everyday Japanese!

---

## 🚀 Get Started in Seconds!

### Method 1: Start with Web (Easiest!)

1. **Double-click** `nadesiko3ai_web_en.html`
2. Browser opens, and you're ready to start!
3. Type commands in simple English in the text box

### Method 2: Start with Command Prompt

1. **Double-click** `nadesiko3ai_starter_en.js`
2. Black screen appears, type commands in English
3. Press Enter to execute!
